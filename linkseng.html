Discord

Discord is an app mostly used by gamers. You can both chat and make phonecalls with someone or with a whole group. Look at it as a more sofisticated and modernised version of Skype.

Stackoverflow

Stackoverflow is an online forum on which you can post IT-related questions. It's up to other users to answer these questions. People then can indicate which answer actually is a solution to the question. Based on those indications the best solution is shown.

W3Schools

W3Schools is briefly said an online course for everything that has something to do with webdesign. From HTML and CSS to Javascript, you can find it all. With each part of the course comes one or more examples that you can alter yourself to find out excactly how it works.

GitHub

GitHub is a website on which software can be posted. Programmers from all over the world can post their code on the site, which then can be used by others. GitHub also provides some sort of Google Docs for programming. People can work on the same code at the same time, which saves a lot of time.
