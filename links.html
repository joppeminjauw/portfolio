Discord

Discord is een app vooral gebruikt door gamers. Je kan er zowel chats als spraakgesprekken met één of meerdere personen mee voeren. Je kan het zien als een beter uitgewerkte en modernere versie van Skype.

Stackoverflow

Stackoverflow is een online forum waarin je informatica gerelateerde vragen kan stellen. Er worden dan oplossingen aangereikt door andere gebruikers. Mensen kunnen aanduiden of de oplossing effectief werkt en op basis van die indicaties wordt de beste oplossing naar voren geschoven.

W3Schools

W3Schools is kort gezegd een online cursus van alles wat met webdesign te maken heeft. Van HTML en CSS tot Javascript, je kan het er allemaal vinden. Bij elk onderdeeltje van de talen staan er ook telkens één of meerdere voorbeelden, die je zelf kan aanpassen om te ondervinden hoe het precies werkt.

GitHub

GitHub is een website waarop software kan geplaatst worden. Programmeurs van overal op de wereld kunnen hier code plaatsen die dan door anderen gebruikt kan worden. GitHub maakt het ook mogelijk voor mensen om tegelijk aan dezelfde code te werken, zodat deze niet telkens manueel doorgestuurd moet worden naar elkaar. Een soort Google Docs voor programmeren.
